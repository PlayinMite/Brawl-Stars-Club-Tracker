# Brawl-Stars-Club-Tracker
An app that helps you keep track of your club!

# Features
• Shows total people, and the amount of normal members, seniors, and vice presidents at a glance.

• Edit existing members.

• Take notes on members, in case you need to remember something specific on particular members (eg. dont kick this guy, he has a valid excuse)

• Keep track of Mega Pig streaks! (This is a thing i see many clubs using as proof of loyalty and shows eligibility for promotions)

Use the info tab in the top left of the app for more info!
